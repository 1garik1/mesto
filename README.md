# Проект: Место
Данное веб-приложение позволяет добавлять понравившиеся фотографии и увеличивать по клику, а также менять имя профиля и его род занятий
### При разработке использоввались:

* grid;
* flex;
* js;
* ООП;
* figma;
* файловая структура 'Nested';
* Картинки.
